1) What are the benefits of using CSS?
ANS) CSS enables precise control over webpage layout, typography, colors, and spacing, enhancing visual appeal and readability. Its flexible nature facilitates responsiveness, ensuring seamless adaptation to diverse devices and screen sizes for optimal user interaction.


2) What are the disadvantages of CSS?
ANS) CSS can pose challenges in cross-browser compatibility, global scope issues, and complexity in achieving intricate layouts. Managing performance impact and debugging inefficiencies can also be problematic, along with the absence of native variable support in older versions.

3) What is the difference between CSS2 and CSS3?
ANS) CSS3 introduces new features like animations, transitions, and flexible box layouts, offering more advanced styling capabilities compared to CSS2. Additionally, CSS3 provides better support for responsive web design and compatibility with modern browsers, enhancing the overall user experience.

ANS) Inline Styles, Internal Stylesheets, External Stylesheets

14) What is embedded style sheets?
ANS) Embedded style sheets refer to CSS styles that are defined within the <style> element and included directly in the <head> section of an HTML document.

15) What are the external style sheets?
ANS) External style sheets are CSS files that are created separately from HTML documents and linked to them using the <link> element. 

16) What are the advantages and disadvantages of using external style sheets?
ANS) Advantages: Ease of Maintenance, Reusability, Consistency
Disadvantages: Specificity and Overriding, File Management, Dependency

17) What is the meaning of the CSS selector?
ANS) In CSS, a selector is a pattern that specifies which elements in an HTML document the associated CSS rules should apply to. Selectors target specific HTML elements based on their type, attributes, classes, IDs, or relationships with other elements in the document structure.
